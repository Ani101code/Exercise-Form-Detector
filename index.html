🎯 Project Aim

The aim of this project is to develop an AI-powered exercise form analysis system that uses computer vision to analyze human movement in real time, provide feedback on exercise technique, and count exercise repetitions automatically.

The system detects body posture from live video input and evaluates joint positions and movement patterns to identify incorrect form, asymmetries, and unsafe movements. It tracks movement cycles to accurately count repetitions (reps), helping users monitor training volume while reducing injury risk and improving exercise efficiency.

✨ Key Features

🎥 Real-time Analysis

Live video input with instant form feedback using MediaPipe pose estimation (33 landmarks). 10-frame median smoothing for accuracy.

📊 Angle Calculation

Precise joint angle measurement (elbow, hip, knee) with vector-based calculation and hysteresis filtering.

🔄 Smart Rep Counting

Automatic repetition tracking with form validation, 1-2 second cooldown, and state machine for each exercise.

⚠️ Form Correction

Identifies incorrect form: sagging hips, kipping, knee valgus/caving, knee flaring, asymmetry, and partial range of motion.

🔊 Audio Feedback

Verbal cues for form correction using TTS (pyttsx3/win32com) with intelligent 4-8 second cooldown system.

📈 Session Tracking

Track reps across push-ups, pull-ups, and squats with individual reset (R) and session reset (T) options.

📋 Strict Form Requirements

Based on biomechanics research and exercise science literature. Reps only count when ALL criteria are met. Thresholds directly from Python code.

💪 PUSH-UP

  • Down position: Elbow < 90°
  • Up position: Elbow > 160°
  • Body angle: 160-180° (straight)
  • Hip ideal: 170° ±15° tolerance
  • No knee pushups: knee > 150°
  • Stand SIDEWAYS to camera
  • Min down time: 0.2s

⬆️ PULL-UP

  • Top position: Elbow < 90° (chin over bar)
  • Bottom position: Elbow > 150° (full hang)
  • No kipping: Hip angle > 150°
  • Asymmetry tolerance: < 25° difference
  • Wrists must be above shoulders
  • Full range of motion required
  • Min top time: 0.3s, min bottom: 0.3s

🦵 SQUAT

  • Up position: Knee > 155° (standing)
  • Down position: Knee < 125° (parallel+)
  • Knee caving in: Knee inside hip margin
  • Knee flaring out: Ratio > 2.2x hip width
  • Max knee width ratio: < 1.6x hip width
  • Hysteresis: 10°
  • Min down time: 0.2s

📝 Notes for Use in This Project

  • Technical thresholds (e.g., elbow/knee/hip angle cutoffs) should be treated as heuristic ranges informed by literature and coaching guidelines, then calibrated empirically for camera setup and user population.
  • EMG findings are exercise-variation dependent (grip width, cadence, tempo, load, depth, fatigue), so references are best used to justify rule directionality (what tends to increase/decrease activation), not as absolute universal cutoffs.
